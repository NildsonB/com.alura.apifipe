
# PROJETO FIPE API - CONSULTA DE VALORES DE VEÍCULOS

Este projeto foi desenvolvido como parte de estudos e consolidação de conhecimentos, proposto no curso "Java: Trabalhando com lambdas, streams e Spring Framework", disponibilizado pela [Alura](https://cursos.alura.com.br/course/java-trabalhando-lambdas-streams-spring-framework). O projeto foi desenvolvido em Java e suas funções são executadas através da linha de comando.
A api utilizada para o desenvolvimento deste projeto foi a [FIPE API HTTP REST](https://deividfortuna.github.io/fipe/?ref=dr-pa&utm_medium=public-apis-website). Uma api de desenvolvida por brasileiros e de uso gratuito.

## 🚀 Funcionalidades
- Escolha de tipo de veículo (carro, moto ou caminhão).
- Seleção de marca, a partir do tipo de veículo escolhido.
- Filtragem pelo nome do modelo desejado.
- Apresentação das avaliações de preço de todos os anos cadastrados do modelo escolhido.

## 🛠️ Tecnologias
- Java 21
- Springboot
- Jackson
- Postman

## 📖 Objetivos
Este projeto não possui fins comerciais, foi desenvolvido para praticar:
- Comsulmo de API's REST
- Manipulação de dados JSON com uso da biblioteca Jackson
- Estruturação de projetos Java
- Uso de Lambdas e Streams
